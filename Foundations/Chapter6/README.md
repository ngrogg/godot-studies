# Chapter 6: Frequently Asked Questions

## Notes
### Navigation
* **How do I navigate my scene?**, Import and install the libraries "3D Car with SettingsPanel" or "Simple First Person Controller".
* **After importing my Simple First Person Controller I can't move around the scene.**, Make sure the Input map is configured so controls are linked to the proper keywords. <br>
  Ex. W to player_forwards. <br>

### Transformations and assets
* **How do I import assets in my scene?**, Import the asset by dragging and dropping asset inside the FileSystem dock in Godot.
* **How can I transform objects?**, Select the object and use the keyboard shortcuts W,E,R,T or modify the objects transform properties in the Inspector window.

### Creating, Organizing, and Searching for objects and assets
* **How do I create an object?**, Select the node and press "Control + A" or right click.
* **How do I add a texture?**, Click on the arrow to the right of the attribute called "Material" for the object. <br>
  In the new contextual menu select the option called "New SpatialMaterial".
* **How do I group objects?**, Create an empty object (parent) and drag-and-drop the objects to be grouped on the parent.
* **How do I look for objects in my project**, Use the project search window and search by name.
* **If I import an asset in my project, can I access it from any scene within this project?**, Yes. This will also save space on the hard drive.
